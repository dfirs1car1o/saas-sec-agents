## 5) Transaction Security Policy Requirements

Impossible travel login patterns
High-volume export anomalies
Suspicious API token and integration usage
Privileged admin risky config changes
- Allowed actions: Block, challenge, notify, allow with audit
- Production approval owner: CorpIS Change Advisory Board

## 6) Identity and Access Baseline
- MFA: MFA required for all interactive and privileged users
- Session settings: Session timeout 30 minutes idle, lock on suspicious activity
- IP/network controls: Trusted IP ranges for admin and integration profiles
- Privileged governance: Quarterly recertification of privileged roles

- Encryption/masking: Encryption at rest and masking for sensitive fields
- Data residency constraints: UK and EU data residency for in-scope business data
